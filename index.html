Make great oldschool games

Akihabara is an open source game development framework for making browser-based 16 bit games.
It uses the best of HTML5 and JavaScript.

Download Latest Stable Build (1.3.2)

Akihabara is an open source project originally created by Francesco Cottone a.k.a. KesieV
